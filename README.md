# CompileFinalWork

## 文法
采用C语言的基础文法，如下：
````
<程序> ::= <声明列表>

<声明列表> ::= <声明> <声明列表> | <声明>

<声明> ::= <变量声明> | <函数声明> | <结构体声明>

<变量声明> ::= <类型> <标识符> [ '[' <常数> ']' ] ';'

<类型> ::= int | float | char | double | struct <标识符>

<函数声明> ::= <类型> <标识符> '(' [ <参数列表> ] ')' '{' <复合语句> '}'

<参数列表> ::= <参数> | <参数> ',' <参数列表>

<参数> ::= <类型> <标识符>

<结构体声明> ::= struct <标识符> '{' <变量声明列表> '}' ';'

<变量声明列表> ::= <变量声明> <变量声明列表> | <变量声明>

<复合语句> ::= '{' <语句列表> '}'

<语句列表> ::= <语句> <语句列表> | <语句>

<语句> ::= <表达式语句> | <复合语句> | <选择语句> | <迭代语句> | <跳转语句>

<表达式语句> ::= <表达式> ';' | ';'

<选择语句> ::= if '(' <表达式> ')' <语句> [ else <语句> ]

<迭代语句> ::= while '(' <表达式> ')' <语句>

<跳转语句> ::= return [ <表达式> ] ';'

<表达式> ::= <标识符> '=' <表达式> | <逻辑表达式>

<逻辑表达式> ::= <逻辑表达式> '||' <逻辑表达式> | <逻辑表达式> '&&' <逻辑表达式> | <关系表达式>

<关系表达式> ::= <关系表达式> '==' <关系表达式> | <关系表达式> '!=' <关系表达式> | <算术表达式>

<算术表达式> ::= <算术表达式> '+' <项> | <项>

<项> ::= <项> '*' <因子> | <因子>

<因子> ::= <标识符> | <常数> | '(' <表达式> ')'

<常数> ::= [0-9]+ ('.' [0-9]+)?

<标识符> ::= [a-zA-Z_][a-zA-Z0-9_]*
````
